Kompetensi yang Dipelajari Selama PKL

Selama melaksanakan Praktik Kerja Lapangan (PKL), saya memperoleh berbagai pengalaman dan keterampilan yang berkaitan dengan bidang Rekayasa Perangkat Lunak. Berikut beberapa kompetensi utama yang saya pelajari:

Flowchart (Diagram Alir)

Mempelajari cara menyusun logika program secara sistematis menggunakan flowchart untuk membantu perencanaan dan perancangan program yang efisien dan mudah dipahami.

Normalisasi Data

Belajar tentang cara mengelola struktur basis data agar lebih teratur dan efisien, termasuk penerapan bentuk normal (1NF, 2NF, 3NF) untuk menghindari redundansi data.

Penggunaan Navicat

Menggunakan aplikasi Navicat untuk mengelola basis data MySQL, membuat tabel, mengatur relasi antar tabel, serta melakukan query data secara langsung.

Dengan kompetensi tersebut, saya semakin memahami bagaimana proses pengembangan perangkat lunak dimulai dari perancangan logika, pembuatan database, hingga implementasi dalam bentuk aplikasi nyata.

Proyek yang Dikerjakan

DELPHI

Selama melaksanakan Praktik Kerja Lapangan (PKL), saya mengerjakan sebuah proyek pembuatan aplikasi sederhana menggunakan Delphi dan bahasa pemrograman Pascal. Proyek ini berfungsi untuk mengelola data siswa dengan fitur seperti input data, pencarian, dan penyimpanan ke basis data yang dirancang menggunakan Navicat. Dalam prosesnya, saya menerapkan konsep flowchart untuk perancangan logika program dan normalisasi data untuk memastikan struktur database yang efisien dan terhindar dari duplikasi.


LARAVEL

Selama melaksanakan Praktik Kerja Lapangan (PKL), saya mengerjakan sebuah proyek berupa website pendaftaran PKL di BBPPMPV BOE Malang menggunakan framework Laravel. Website ini berfungsi untuk mempermudah proses pendaftaran peserta PKL dengan fitur seperti input data siswa, validasi formulir, penyimpanan data ke dalam basis data, serta tampilan daftar pendaftar yang dapat diakses oleh admin. Basis data dikembangkan menggunakan MySQL dan dikelola melalui Navicat untuk mempermudah pengaturan tabel serta relasi antar data. Dalam proses pengembangannya, saya juga menerapkan flowchart sebagai acuan logika sistem dan normalisasi data untuk memastikan struktur basis data yang efisien serta bebas dari duplikasi.



FLOWCHART

Dalam proyek ini, saya mempelajari cara membuat alur logika suatu sistem menggunakan flowchart. Flowchart digunakan untuk menggambarkan urutan proses atau langkah-langkah yang terjadi dalam suatu program secara terstruktur dan mudah dipahami. Melalui pembuatan flowchart, saya belajar bagaimana mengidentifikasi proses utama, menentukan keputusan yang memengaruhi jalannya program, serta menyusun hubungan antarproses dengan simbol-simbol standar. Penerapan flowchart membantu saya memahami dasar logika pemrograman sebelum menulis kode, sehingga dapat meminimalkan kesalahan, memperjelas dokumentasi sistem, dan mempermudah proses pengembangan aplikasi.



FIGMA

Dalam proyek ini, saya menggunakan Figma untuk membuat desain antarmuka (User Interface) website pendaftaran PKL di BBPPMPV BOE Malang. Melalui proyek ini, saya mempelajari cara merancang tampilan website yang menarik, responsif, dan mudah digunakan. Saya menyusun layout halaman seperti beranda, formulir pendaftaran, dan halaman admin dengan memperhatikan prinsip UI/UX Design agar pengalaman pengguna menjadi lebih nyaman dan efisien. Selain itu, saya juga belajar membuat prototipe interaktif di Figma untuk memvisualisasikan alur navigasi dan interaksi antarhalaman sebelum website dikembangkan menggunakan framework Laravel. Dengan demikian, proyek ini membantu saya memahami pentingnya tahap perancangan visual dalam proses pengembangan aplikasi berbasis web.
